# Off the Dial

A first-person 3D puzzle-horror game set in a cursed, fog-covered town. Solve riddles, explore interconnected locations, and survive a relentless creature that hunts by sound and light.

## About

You awaken in an abandoned town with no memory of how you arrived. Strange symbols mark crumbling buildings, sealed pathways block every route out, and something is watching. The only way to escape is to explore the town's interconnected zones — solving environmental puzzles while staying silent enough to avoid the creature that roams freely across the map.

Every puzzle solved brings you closer to escape. Every mistake alerts the creature to your presence.

## Features

- **First-person exploration** across a fog-covered town with multiple distinct zones
- **Environmental puzzles** themed to each location — medical puzzles in the hospital, pipe-routing in the tunnels, and more
- **Adaptive creature AI** that learns your hiding spots and adjusts aggression based on your puzzle speed
- **Sound-based detection** — the creature is drawn to noise and light, making stealth essential
- **Sanity system** — as sanity drops, puzzle text scrambles and the environment distorts
- **Cross-zone progression** — some puzzles require items found in different locations, encouraging backtracking
- **Performance ratings** from D to S rank based on time, encounters survived, and collectibles found

- **Engine:** Unity 6 (6000.3.9f1 LTS)
- **Render Pipeline:** Universal Render Pipeline (URP)
- **Platform:** PC
- **Language:** C#

## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/<your-username>/off-the-dial.git
   ```
2. Open the project in Unity Hub (Unity 6000.3.9f1 LTS)
3. Let Unity import all assets and resolve packages
4. Open `Assets/Scenes/SampleScene.unity` to get started
